== Description ==

= General =

sxss Wiki generates a linked list of content from your posts or pages. The plugin parses the content of posts and pages for pre defined tags like 'h1', 'h2', 'h3', 'h4', 'strong', or 'em', makes them linkable and generates the linked list to this segments at the top of your post or page.

= Translation =

German translation will come in the next days.

= Known problems =

If you have the same headline twice on a page, the link in the index list will only link to the first headline on the page.

= Misc =

More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Upload the sxss Wiki plugin to your blog, activate it. You will now see the sxss Wiki box on your post and pages edit screen
